# Movie Search App

## Description
Movie Search App is an interactive web application that allows users to search for movies, view details, and explore trending films. It demonstrates API integration, dynamic UI updates, and responsive design. This project showcases front-end development skills with real-world functionality.

## Features
- Search for movies by title
- View movie details (rating, release date, overview)
- Responsive design for desktop and mobile
- Smooth transitions and interactive UI
- API integration with movie database (e.g., TMDB)
